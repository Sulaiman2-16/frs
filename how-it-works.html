How does facial recognition system work?

There are so many different algorithms of facial recognition system build to compare faces from images and identify the person from the database. Sometimes the algorithms compare shape of the eyes, cheekbones, nose, or jaw and sometimes they compare all of these features at ones. Than the computer will find a match and returns an information about the person. All of this happened in less than a second and it is very accurate. Some systems will have near 100% accuracy.
